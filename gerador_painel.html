Publicar direto no GitHub (opcional)

Publica o index.html direto no seu repositório, sem precisar baixar e subir manualmente. Precisa de um Personal Access Token do GitHub (veja como criar abaixo). O token fica salvo só no seu navegador — nunca passa por mim.

Como criar o token (uma vez só):
1. Acesse github.com/settings/personal-access-tokens/new
2. Em "Repository access", escolha "Only select repositories" → selecione o seu repositório
3. Em "Permissions → Repository permissions", defina Contents: Read and write
4. Clique em "Generate token" e cole o valor aqui em cima

Tudo roda no seu navegador — a planilha não sai do seu computador.
Depois de gerar, baixe o arquivo ou publique direto no GitHub acima.
